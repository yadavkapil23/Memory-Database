## 🆘 Getting Unstuck

**If Stuck on Code:**
1. Run tests to see specific error
2. Check code docstring
3. Review relevant section in technical design
4. Check `04_ARCHITECTURE_REFERENCE.md`
5. Search code comments for hints

**If Stuck on Performance:**
1. Run profiling script
2. Identify bottleneck
3. Review optimization section in weekly guide
4. Measure improvement
5. Compare to targets

**If Stuck on Testing:**
1. Review existing tests
2. Understand test pattern
3. Write similar test
4. Run and verify
5. Document approach

**If Major Blocker:**
1. Document the issue clearly
2. Review relevant documentation
3. Check code comments
4. Try alternative approach
5. Continue other tasks (don't get stuck)
