================================================================================
SYSTEM ARCHITECTURE
================================================================================

Three-Layer Memory Hierarchy:

  Working Memory (Current Context)
    ↓ Consolidation Job (Async)
  Episodic Memory (Events) + Forgetting Curve
    ↓ Pattern Extraction
  Semantic Memory (Patterns & Facts)

Key Algorithms:

  Importance Score:
    = 0.20 * Novelty
    + 0.30 * Task_Success
    + 0.25 * Retrieval_Frequency
    + 0.15 * User_Signal
    + 0.10 * Emotional_Salience

  Forgetting Curve:
    Memory_Strength(t) = e^(-λt / Stability)
    Where λ varies by importance tier

Vector Operations:

  Embedding: OpenAI API (3072 dims) with Redis cache
  Similarity: Cosine distance in Qdrant
  Retrieval: Top-K search with ranking
